Loom in Obsidian

Default hotkeys:
- `Ctrl+Space` - complete  
- `Ctrl+Shift+Space` - create child  
- `Alt+Down` - switch to next sibling  
- `Alt+Up` - switch to previous sibling  
- `Alt+Left` - switch to parent  
- `Alt+Right` - switch to (most recently visited) child  
- `Alt+Backspace` - delete current node
